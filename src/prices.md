## Managed hosting (static sites)

You can host up to 20 websites with me for **£40** per month, or **£20 if discounted**. This is a hands-off package - I take care of the technical side so you can focus on your business:

- **Your sites stay online** - I monitor uptime and fix problems
- **Your content is safe** - automatic backups stored in two separate locations
- **Quick changes included** - email me updates and I'll make them for you
- **Bugs get fixed** - if something breaks, I sort it out
- **Marketing support** - personal help implementing strategies from my [free guides](/guides/) and [videos](/videos/)

You always have the option to host elsewhere since you own the full source code. I recommend [Bunny.net](https://bunny.net/), [Netlify](https://netlify.com), [Pico.sh](https://pico.sh/pgs), and [Surge](https://surge.sh) - but I'll help you host wherever you like.

## Basic hosting (unmanaged sites)

If you update your site rarely, or prefer to handle changes yourself, this lighter package is **£10** per month or **£5** if discounted:

- **Your site stays online** - hosted and backed up
- **Bugs get fixed** - if something breaks, I sort it out

You won't get personal marketing support or free content changes - but you can edit your site yourself, use my [free marketing guides](/guides/) and [videos](/videos/), or pay me or my brother Kevin at [KevinBurkeServices.com](https://kevinburkeservices.com) for content changes as needed.

## Managed hosting (web applications)

For custom web applications (booking systems, databases, business tools), I host your app on its own dedicated server. Server costs start at around £5 per month, and I charge **£60** per month (or **£30 if discounted**) on top of that. You get full access to the server and its configuration. This includes:

- **Your app stays online** - I monitor uptime and fix problems
- **Your code is safe** - automatic backups stored in two locations
- **Your database is backed up daily** - and emailed to you for extra safety
- **Quick changes included** - I make small updates at your request
- **Marketing support** - personal help implementing strategies from my [free guides](/guides/) and [videos](/videos/)

You can also host the application yourself, since you own the full source code. I'm happy to help you get set up on any host.

## Emails & domain renewals

I will help you source the ideal provider for your needs and budget. You will register and pay them directly, unless you _really_ don't want to, in which case we can sort something out.

I use [Purelymail](https://purelymail.com) for my own email hosting - they're very affordable and flexible. I can help you connect any email provider to your domain, and I'll run deliverability tests to make sure your emails get through.
