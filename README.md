# www.chobble.com

This is the code behind the site at [www.chobble.com](https://www.chobble.com).

It's an Eleventy site, built with GitHub Actions, and hosted on Bunny.net.

If you want a site like it, you'd be better served by cloning the [Chobble Template](https://www.chobble.com/services/chobble-template) - which is like this site but with a load of cool bells and whistles to make it easy for small businesses to get up and running
